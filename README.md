# Missile Guidance: PD vs Proportional Navigation

This project presents a comparison between two missile guidance techniques implemented using MATLAB and Simulink:

* **Proportional Navigation (PN)**
* **PD Controller**

The objective is to compare the interception performance of both guidance laws against a moving target through simulation.

## Project Files

* `PN_guidance.m` – MATLAB implementation of the Proportional Navigation guidance law.
* `Missile_Guidance.slx` – Simulink implementation of the PD guidance controller.
* `pn_pd_animation.m` – MATLAB script for comparing both guidance methods with animation.

## Features

* Target interception simulation
* Comparison of PD and PN guidance laws
* Missile and target trajectory visualization
* Animation of the interception process
* MATLAB and Simulink implementation
